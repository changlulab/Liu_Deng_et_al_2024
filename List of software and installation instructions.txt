Software used in this work are standard for ChIP-seq data analysis. Without additional note, they are installed in Linux. Command line installation through programs like bioconda or miniconda is preferred to avoid issues with manual configuration and dependency.
1. Trim Galore! 0.4.1
https://bioconda.github.io/recipes/trim-galore/README.html
2. bowtie 1.1.2
http://bioconda.github.io/recipes/bowtie/README.html
3. samtools 1.3.1
https://bioconda.github.io/recipes/samtools/README.html
4. bedtools 2.29.2
https://bioconda.github.io/recipes/bedtools/README.html
5. MACS2 2.1.1
https://bioconda.github.io/recipes/macs2/README.html
6. epic2 0.0.41
https://bioconda.github.io/recipes/epic2/README.html
7. IGV 2.11.2 (Windows)
https://software.broadinstitute.org/software/igv/2.11.x
This software is downloaded and installed on local PC for visualizing and manually inspecting .bw genome browser track files.
8. DiffBind 3.14 (RStudio)
https://bioconductor.org/packages/release/bioc/html/DiffBind.html
This software is installed in RStudio to obtain consensus peakset.
9. deepTools 3.5.0
https://bioconda.github.io/recipes/deeptools/README.html
10. stats 3.6.2 (RStudio)
The "kmeans" function used for clustering is part of this package. It is pre-installed in the standard R package.
11. GREAT 4.0.4 (web)
http://great.stanford.edu/public/html/
This is a browser based software for analyzing GO terms from a list of peaks.
12. Intervene 0.6.1
https://bioconda.github.io/recipes/intervene/README.html
13. hisat2 2.2.1
https://bioconda.github.io/recipes/hisat2/README.html
14. SeqMonk 1.47.1 (Windows)
https://www.bioinformatics.babraham.ac.uk/projects/seqmonk/
15. Deseq2 1.36.0 (RStudio)
https://bioconductor.org/packages/release/bioc/html/DESeq2.html
